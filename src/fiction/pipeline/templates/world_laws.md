## DOCUMENT 1: WORLD BIBLE

### Premise
Cultivation stages map to system architecture evolution. "Spiritual power" is throughput — the capacity to process events from heaven and earth. "Techniques" are consumer processes. "The Dao" is the system's emergent behavior, not a will.

### The Six Stages

| Stage | Systems Concept | Cultivation Metaphor | Core Truth | Failure Mode |
|---|---|---|---|---|
| **Body Refinement** | Decentralization | Body as village, not throne room | You are not one thing | Forcing central will on local resistance → rupture |
| **Qi Condensation** | Protocol design | Organs speak different languages | Translation is not free | Format mismatch → translator death → internal conflict |
| **Foundation Establishment** | Replication & consensus | Five elements = five copies of will | Agreement takes time | Network split → split-brain → body wars with itself |
| **Core Formation** | Leader election & strong consistency | Golden Core = temporary coordinator | Speed has a cost | Bottleneck → lock → system halt under load |
| **Nascent Soul** | Choreography | Hand moves because eye saw | Let go of the vote | Temporal coupling → freeze between event and response |
| **Spirit Severing** | Geographic distribution & quorum | Three fragments, any two = self | Identity is agreement | Partition → divergence → two people, one name |

### Heaven and Earth as System
- Heaven emits events (tribulations, spiritual energy, natural law)
- Earth provides storage (ley lines, spiritual veins, material foundation)
- Cultivators are **consumers** in this stream
- "Lightning tribulation" = sudden traffic surge with unique signature
- "Dao comprehension" = learning the event schema heaven uses

### Magic System Rules
1. **Qi is information carrying energy.** It can be lost (heat dissipation), corrupted (noise), delayed (latency), or duplicated (replay).
2. **Techniques are idempotent or they are dangerous.** A non-idempotent technique invoked twice produces undefined behavior.
3. **Meridians are message buses.** They have throughput limits. Blockage is backpressure. Rupture is buffer overflow.
4. **The dantian is not special.** It is merely a convenient co-location point, not a metaphysical center.
5. **Consciousness is a consistent read.** Not a location. The "self" is what emerges when distributed nodes agree sufficiently.
